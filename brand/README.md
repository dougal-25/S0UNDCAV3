# Sound Cave — Brand Assets

Visual brand assets for The Sound Cave. Not code — just art, logos, and reference material for review.

## Structure

- `logo/` — active logo files (SVG preferred, dated filenames so we keep history)
- `logo/dormant/` — alternates, drafts, or retired logos not currently in use. Kept for reference and possible revival.

## Palette (locked, from `tokens.css`)

| Hex | Role |
|---|---|
| `#0a0a0a` | Cave black (primary bg) |
| `#120e0c` | Warm cave black |
| `#e8e8e8` | Off-white (text / mark) |
| `#888888` | Muted grey |
| `#4a4a4a` | Faint grey |
| `#ff4500` | Orange-red accent (hero) |
| `#ff6a1f` | Hot orange (hover) |
| `#aa2a00` | Deep ember (shadow) |

## Type

- **Display / logo:** DM Mono (monospace, wide tracking `0.18em`, lowercase)
- **Body:** DM Sans
